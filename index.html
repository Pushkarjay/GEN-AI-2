🎯 Key System Features

  • Real-time Updates: 3-second WebSocket broadcasts to all connected clients
  • Multi-Mode System: Real-time + 3 demo scenarios (Peak, Low, Evacuation)
  • Sensor Integration: IR, Pressure, Camera, RFID sensor data aggregation
  • Dynamic Alerts: Occupancy-based thresholds with mode-specific scenarios
  • Queue Intelligence: Automatic queue calculation from zone occupancy
  • Redundancy: WebSocket with HTTP polling fallback for reliability
  • Responsive UI: All 4 pages update simultaneously on data arrival
  • Offline Support: Service Worker enables offline functionality
